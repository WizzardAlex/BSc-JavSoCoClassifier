# Conclusion structure

  * Best setting
    * Reasoning
    * Interpretate the metrics
  * Best feature representation
    * Reasoning
    * Interpretate the metrics
* Sub-RQ:2
  * Ratio vs absolut size

* RQ:1
  * Which classifier performs the best

    * Circumstances, some performs better under certain situation
  * Conclusion

* Problems

* Limitations
* Future work
* Concluding remarks
